# CineMaster - Projet Framework JS (L3)

Ce dépôt contient le projet pratique pour le cours de **Frameworks JavaScript**. Choisir le stack : **React** ou **Vue.js**.

## Structure du projet
- `instructions.md` : Le guide complet de l'exercice avec les étapes à suivre.
- `starters/` : Dossiers de départ contenant la configuration de base (Vite).
    - `react-cineapp/` : Starter pour React.
    - `vue-cineapp/` : Starter pour Vue.js.

## Comment commencer
1. Lisez attentivement le fichier `instructions.md`.
2. Choisissez votre framework.
3. Allez dans le dossier correspondant (`cd starters/react-cineapp` ou `cd starters/vue-cineapp`).
4. Installez les dépendances : `npm install`.
5. Lancez le serveur de développement : `npm run dev`.

## Objectif final
Une application web permettant de rechercher des films, de voir leurs détails et de gérer une liste de favoris persistante.
